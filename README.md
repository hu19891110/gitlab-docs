# GitLab Documentation

This site is generated using [Nanoc](http://nanoc.ws).

## Development

To set this up:

- `bundle install`
- `bundle exec nanoc live`

Changes will be reloaded automatically using [Guard Nanoc](https://github.com/guard/guard-nanoc).
